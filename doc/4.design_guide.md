# Kernel Design System Guide

**Version**: 2.1
**Target**: AI Coding Assistant / Frontend Developers
**Framework**: TailwindCSS v3.4+ / shadcn/ui

---

## **Part 1: Global UX Principles (交互與體驗準則)**

此章節定義 Kernel 的互動靈魂與用戶體驗目標。無論使用何種主題，以下原則皆不可違背。

### **1. 沉浸式心流 (Immersive Flow)**
*   **原則**: 介面應「安靜」且「退後」。
*   **實踐**: 
    *   除核心操作 (CTA) 外，避免使用高飽和度色彩。
    *   內容區 (Editor/Content) 應占據 80% 以上視覺權重，側欄與工具列在不使用時應視覺弱化。

### **2. 觸感與反饋 (Tactile Response)**
*   **原則**: 軟體應具備物理世界的「確認感」，消除數位操作的空虛感。
*   **實踐**:
    *   **Hover**: 所有可互動元件在滑鼠懸停時必須有 *即時* 的狀態變化 (背景色變深/淺、邊框顯現)。
    *   **Active**: 點擊瞬間應有 `scale-95` 或 `brightness-90` 的微小回饋。
    *   **Focus**: 鍵盤操作時，Focus Ring 應清晰但不刺眼 (使用 Ring-offset)。

### **3. 預期性與一致性 (Predictability)**
*   **原則**: 不要讓用戶猜測。
*   **實踐**:
    *   相同功能的元件 (如「新增按鈕」) 在全站應保持絕對一致的位置與樣式。
    *   轉場動畫 (Transitions) 應短促且一致 (150ms - 200ms ease-out)。

## **Part 2: Default Theme - "Reflective Dawn" (預設主題)**

此主題靈感源自 **「黑夜中的黎明 (Dawn in the Dark Night)」**。
它不追求絕對的黑，而是營造一種在深夜中看見第一道微光的**舒適感**與**希望感**。

### **1. 色彩系統 (Color System)**

#### **Atmosphere (氛圍與背景)**
*   **Night (`bg-background`)**: `#09090B` (Zinc-950) - 深邃、寧靜的黑夜基底。
*   **Shadow (`bg-surface`)**: `#18181B` (Zinc-900) - 用於卡片與側欄，像是在黑暗中被微光照亮的物體輪廓。
*   **Dawn (`accent-glow`)**: **"Morning Gold" Gradient**
    *   一種溫暖、擴散的琥珀金光 (Amber-500 to Orange-400)，但透明度極低 (Low Opacity)，避免刺眼。
    *   *Concept*: 就像地平線上的微光，提供方向但不干擾黑暗的寧靜。

#### **Typography Colors (資訊層級)**
*   **Primary (`text-primary`)**: `#F4F4F5` (Zinc-100) - 清晰的訊息，如同星光。
*   **Muted (`text-muted-foreground`)**: `#71717A` (Zinc-400) - 像是在陰影中的細節。

#### **Accent & Highlights (黎明之光)**
*   **Primary Token**: `text-amber-200` / `bg-amber-500/20` (低反差背景)
*   **Usage**:
    *   **Active States**: 選中的項目邊框、進度條。
    *   **Interactive**: 按鈕 Hover 時隱約透出的金色光暈。

### **2. UI 構造 (UI Construction)**

#### **Ambient Material (氛圍材質)**
*   **原則**: 結合 **Matte (啞光)** 實體與 **Diffuse (漫射)** 光影。
*   **Borders**: 
    *   極細微的邊框 `border-white/5`。
    *   **Active Border**: 當元件 Active 時，邊框帶有微弱的金色漸層光暈 (`border-amber-500/30`)。
*   **Depth**: 
    *   不使用強烈的 Drop Shadow。
    *   使用 **Inner Glow** 或 **Gradient Mask** 來模擬光線從底部或邊緣滲出的感覺。

#### **Typography (人文氣息)**
*   **Serif Headers**: 使用 `Newsreader` 或 `Playfair Display` 於大標題，傳遞像書籍般的經典質感。
*   **Ui Font**: `Inter` 用於功能性文字，保持極高的可讀性。

### **3. 元件樣式 (Component Styles)**

#### **Cards & Surfaces**
*   **Look**: 深色啞光表面，看似平坦，但在邊緣有極微弱的光線反射。
*   **Interaction**: Hover 時，卡片不只是變亮，而是彷彿被一道暖光掃過 (Subtle gradient overlay)。

#### **Navigation & Sidebar**
*   **Style**: 完全消融於背景中 (Transparent / Blended)，僅 Icon 與文字浮現。
*   **Active Item**: 左側亮起一道細微的金色光條 (Indicator)，背景帶有極淡的琥珀色洗鍊 (`bg-amber-500/5`)。

### **4: AI 實作策略 (AI Implementation Strategy)**

為確保所有生成的 UI 一致呈現 **"Reflective Dawn"** 風格：

> "Adopt the **'Reflective Dawn'** theme."
> "Key Vibe: **'Dawn in the Dark Night' (黑夜黎明)** - Comfortable, Warm, Hopeful."
> "Colors: **Zinc-950** (Night) background. Accents are **Diffuse Amber/Gold** (Dawn)."
> "Lighting: Use **ambient interior lighting** logic. No harsh neons. Buttons glow softly."
> "Typography: **Serif** for headings (Humanist), **Sans-serif** for UI."
> "Components: Dark matte surfaces with **subtle golden borders** on interaction."
